# 유저 생성

주문을 하기에 앞서 여러 대의 키오스크에서 주문하는 것을 고려하여 주문을 시작한 유저 데이터를 먼저 생성합니다. 이후 주문할 때 발급 받은 유저 id값을 이용하여 주문을 해야 합니다.

|기능|<center>URL</center>|메소드|Body|Return|
|:----:|:------|:-----:|:---:|:---:|
|[유저 생성]()|/first|GET|X|JSON|


## Example

### Return
- 유저의 id값을 JSON 형식으로 return합니다.

```json
{
    "id": 1
}
```
